## 🚀 구현한것들
세 가지 case 분류  

1️⃣ 피어(Peer)  
  - 지갑 생성
  - Transaction 전송
  - Transaction 리스트  


2️⃣ 마이너(Miner)  
  - Transaction 보기
  - Merkle Tree
  - nonce 계산
  - 블록 생성(마이닝)  


3️⃣ 블록체인 네트워크(Network)
  - 블록 검증(by nonce 값)
  - 블록 체인 보기


## 🛠 기술 스택
- ### **사용 언어**
java

- ### **개발 환경**

Spring : 서버 구축  

Thymeleaf : 템플릿 엔진(View)

- ### **사용 라이브러리**
security : SHA256, GenerateKeyPair 등 암호화 관련 라이브러리  

base64 : encoders and decoders for the Base64
